◆ initiateApp()

void main.presentation.CtrlApplication.initiateApp ( )

Funció que inicialitza l'aplicatiu amb la còpia de seguretat i programa un backup periòdic.

Amb aquest mètode es permet inicialitzar l'aplicatiu a partir de l'última còpia de seguretat de què es disposi, i a més programa l'autosalvat del sistema cada 120 segons, tot i que es fa al tancar l'apliació, per seguretat

Precondició
Si existeix una còpia de seguretat, aquestà és vàlida
Postcondició
El sistema queda inicialitzat amb la darrera còpia de seguretat del sistema, i a partir d'ara cada 120 segons el sistema s'autoguardarà

void main.presentation.CtrlApplication.closeApp ( JFrame  reference)

Funció que deixa l'estat del sistema en una còpia de seguretat.

Aquest mètode s'ha de cridar abans de tancar l'aplicatiu per tal de salvar-ne el sistema localment i poder-lo restablir en la següent posada en marxa

Paràmetres
referenceFrame de referència per saber on posicionar el dialog en cas d'error
Postcondició
El sistema no queda alterat, però es genera una còpia de seguretat a partir de l'estat actual, que queda emmagatzemada
