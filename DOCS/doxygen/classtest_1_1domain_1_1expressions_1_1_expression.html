Funció per imitar la creació d'una expressió

En la classe de veritat, Expression és abstracte i té aquest mètode per donar-ne d'alta una. Aquí el declarem també, amb la possbilitat de llençar l'excepció de invalid (una possibilitat en la classe de veritat) quan rebem el string "invalid", per poder fer proves

Paràmetres
expressionamb el string que identificaria l'expressió
Retorna
Expressió booleana per poder provar, amb les funcionalitats bàsiques. En cas que el paràmetre sigui "true", es retorna una instància de True. En cas que el paràmetre sigui "false", una de False. Altrament, una Expression de cap de les dues subclasses.
Excepcions
ExceptionInvalidExpressionen cas que el string donat digui "invalid"
