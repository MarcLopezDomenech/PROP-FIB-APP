Mètode per donar d'alta un nou document.

L'operació rep el nom i el títol del nou document a donar d'alta a l'aplicatiu, i sempre que no n'existeixi un amb els mateixos identificadors, així ho fa amb contingut buit.

Paràmetres
titleTítol del nou document
authorAutor del nou document
languageIdioma del nou document
Postcondició
Existeix a l'aplicatiu un document identificat per (title, author). Si no existia prèviament, el contingut serà buit.
Excepcions
ExceptionDocumentExistsen cas que ja existeixi un document identificat per (title, author) a l'aplicatiu
ExceptionInvalidLanguageen cas que l'idioma introduït no sigui "ca", "en" ni "es"

Mètode per importar documents al sistema.

A partir d'un path absolut, es carrega el document al sistema, assignant-li l'idioma donat

Precondició
El path donat té extensió .txt, .xml o .fp
En el text pla, com a mínim la primera línia contindrà el autor, la segona el títol i a partir de la tercera hi serà el contingut. En xml, hi haurà com a mínim etiquetes títol, autor i contingut.
L'idioma del paràmetre és "ca", "en" o "es"
No existeix un document amb el títol i autor que identifiquen el document que es vol importar
Paràmetres
pathRuta al document que es vol donar afegir al sistema
languageIdioma que se li vol assignar al document a importar
Retorna
Objecte amb (favorit, títol, autor) del documet importat
Postcondició
El sistema té un nou document donat d'alta, que és el que hi havia en el path donat i té com a idioma el donat
Excepcions
ExceptionInvalidFormaten cas que el path no tingui extensió .txt, .xml o .fp
FileNotFoundExceptionsi no es pot accedir a la path donada, per permisos o perquè no existeix
ExceptionDocumentExistsquan el sistema ja té donat d'alta un document amb els títol i autor del document que es vol importar
ExceptionInvalidLanguageen cas que l'idioma del paràmetre no sigui ni "ca" ni "en" ni "es"
